[진행중]영화관리/예매 시스템

  • 호스팅링크 / github

    프로젝트로이동 github로이동
  • 개발환경
    Java1.8 ,Spring , mybatis, Mysql , Tomcat8.0, Javascript, jQuery, ajax, materialize, bootstrap, semantic ui.
  • 설명
    4인이서 진행한 팀 프로젝트
  • 프로젝트중 담당한 부분

    클라이언트 페이지 화면 구현 및 메뉴 모듈 분할
    메가박스의 디자인과 인터렉션을 그대로 구현하기 위해
    3가지의 css 프레임워크를 사용하였다.
    프레임워크의 함수가 충돌하는 부분에서는
    컴포넌트 단위로 나누어 사용하거나 iframe을 사용했다.

    상영일정 선택
    이번 프로젝트에서 내가 담당한 부분중에 개인적으로 가장 난이도가 높았던 부분이다
    상영일정 선택 부분에서는 영화,지점,날짜,시간 이라는 4가지의 데이터가 조건과 값이된다.
    조건을 선택하면 조건에 의한 조건이 리프레시 되야한다.
    조건에 의한 값과 조건이 같이 변경되어 무엇을 클릭하더라도 대처할수 있도록
    프론트엔드 와 백엔드에서 양방향으로 분기문을 여러가지 경우의수로 나누었고
    조건에 따른 동적쿼리를 사용하였다.

    좌석 선택
    상영정보 및 현재 예매가 된 좌석이나
    예매가능한 좌석들을 보여주며 선택할수 있게 한 페이지이다.
    특정 지점의 특정 상영관은 행 좌석수와 열 좌석수를 가지고있다.
    행*열 만큼 for문을 사용하여 좌석을 배치했고
    좌석은 각자 자기자신의 좌석 코드를 가지고있도록 구현하였다.

    영화 리스트
    현재 상영중인 영화의 관람객수 , 개봉일 , 개봉상태 를 기준으로
    박스오피스,최신개봉작,상영예정작 으로 정렬할수 있게 구현하였고
    평점을 볼수 있고 로그인시 좋아요를 할 수 있게 하였다.

    영화 상세
    영화의 상세 정보나 평점 좋아요수 등을 볼수 있고
    감독이나 출연진을 통해 인물 페이지로 이동할 수 있다.
    예매 비율 부분은 구글 차트를 이용하였다.

    영화 상세
    한줄평과 평점을 등록하면 영화 통계에 반영이 된다.

    인물 상세
    인물의 상세 정보를 볼수 있고
    필모그래피를 통해 제작하거나 출연한 영화의 상세페이지로 이동할 수 있다.
